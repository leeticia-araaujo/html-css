Aprendendo a utilizar Media Queries

Media Queries são uma técnica do CSS que permite aplicar estilos diferentes dependendo das características do dispositivo, como largura da tela, orientação e resolução. Elas são essenciais para criar layouts responsivos que se adaptam a diferentes tamanhos de tela, garantindo uma melhor experiência de usuário em dispositivos móveis e desktops.

Para utilizar Media Queries, você pode definir regras CSS dentro de uma consulta que verifica as condições do dispositivo. Por exemplo, para aplicar estilos apenas em telas com largura máxima de 600px, você pode usar a seguinte sintaxe: 

                @media (max-width: 600px) {
                    body {
                        background-color: lightblue;
                    }
                }

Neste exemplo, quando a largura da tela for de 600 pixels ou menos, o fundo do corpo da página será alterado para azul claro. Você pode combinar várias condições e aplicar diferentes estilos para criar um design flexível e adaptável.
