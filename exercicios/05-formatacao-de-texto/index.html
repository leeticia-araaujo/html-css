Principais formatações de texto

Negrito

Nessa frase temos um termo em destaque/negrito usando a tag < b > (de forma não semântica)

Já nessa frase o termo em destaque usa a tag < strong > para ficar destacado de forma semântica, com significado pro html

Itálico

Aqui temos um termo em itálico usando a tag < i > (não semantica)

Já nessa frase o termo em itálico tem semântica, usando a tag < em >

Texto marcado

Podemos criar um texto marcado usando a tag < mark >
O texto inicialmete fica marcado em amarelo, mas a cor pode ser aterada usando o css.
Em css usamos o parâmentro style + backgroud color para definir a cor da marcação.
Podemos fazer isso dentro da tag ou ddentro do head. Caso seja na tag só vai mudar aquela em específico, caso seja no head vão ser todas as tags(de uma mesma pág)

Texto grande e texto pequeno

Neste parágrafo temo um texto "grande" e um texto "pequeno" usando as tags < big > e < small >, podemos mudar o tamanho da letra usando apenas o css

Texto deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido mas não considerado. Para isso usamo a tag < del >

Texto inserido

Podemos marcar um texto como inserido usando a tag < ins > para dar ênfase e indicar que ele foi adicionado depois

Existe também a tag < u > que deixa o texto sublinhado, mas sem semântica

Texto sobrescrito

Para inserir coisas do tipo "x20+12" usamos a tag < sup >

Texto subscrito

Para inserir coisas do tipo "H2O" "CO2"usamos a tag < sub >

Obs

Semântica: É o estudo do significado. A semântica linguística estuda o significado usado
por seres humanos para se expressar através da linguagem. Ela dâ significado

As tags usadas e que estão sublinhadas no código são tags obsoletas.
