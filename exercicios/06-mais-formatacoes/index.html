Código-fonte

O comando document.getElementById('test') é escrito em lingaguem JavaScript, usamos a tag < code > (but ela não identa o código)

Código em JavaScript usando a tag < code >:
var x = 10; if (x > 10) { console.log("X é maior que 10") } else if (x < 10) { console.log ("X é menor que 10") } else { console.log("x é igual a 10") }

Colocando a tag < pre > o código que estiver dentro dela vai ficar igual foi escrito. A tag pre considera tudo como está dentro dela. 


    var x = 10;
    if (x > 10) {
        console.log("X é maior que 10")
    } else if (x < 10) {
        console.log ("X é menor que 10")
    } else {
        console.log("x é igual a 10")
    }

Texto Invertido

Usando a tag < bdo > deixamos o texto invertido. Com o parâmetro dir="rtl" o texto fica na direção direita para esquerda. Usando o parâmetro dir="ltr" o texto fica na direção esquerda para direita.
